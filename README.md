## How to use!
Delays - Calculates how long it should take for the grind to continue.



Keybinds - The keys for the grinding.



Controls - Helps you know which keys for AFK grinding without overriding the built-in Antimatter Dimensions keybinds.

## Keybind Usage

Run Script + Apply Settings (Q)

Stop Script (E)

Exit Script / Close Script (Z)



Shift+1: Toggle Max All

Shift+2: Toggle Hold Crunch

Shift+3: Toggle Auto Tab

Shift+4: Toggle Rebirths
